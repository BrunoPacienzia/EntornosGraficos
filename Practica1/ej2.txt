2.c

Elemento:
    <img src="" alt="lugar imagen" id="im1" name="im1" width="32" height="32" longdesc="detalles.htm" />

Efecto:
    Inserta una imagen

Seccion:
    body.

Etiquetas:
    <img ..../> (obligatorio)

Atributos:

    src: indica la ruta de la imagen, valor = "" (obligatorio)
    alt: indica texto alternativo, valor= lugar imagen (opcional)
    id: para identificación, valor = im1 (opcional)
    name: nombre, valor = im1 (opcional)
    width: ancho, valor = 32 (opcional)
    height: alto, valor = 32 (opcional)
    longdesc: descripcion larga, valor = detalles.htm (opcional)


2.d

Elementos:
    <meta name="keywords" lang="es" content="casa, compra, venta, alquiler " />
    <meta http-equiv="expires" content="16-Sep-2019 7:49 PM" />

Efecto:
    Define cuales son las palabras clave del sitio y cuando expira el contenido de la misma.

Seccion:
    Head

Etiquetas:
    <meta ..../>

Atributos:
    name: indica el tipo de metadato, valor = keywords (opcional)
    lang: indica el idioma del contenido, valor = es (opcional)
    content: define el valor del metadato, valor = casa, compra, venta, alquiler (obligatorio)
    http-equiv: simula cabeceras de respuesta HTTP, valor = expires (opcional)
    content: define el valor del metadato, valor = 16-Sep-2019 7:49 PM (obligatorio)


2.e

Elemento:
    <a href="http://www.e-style.com.ar/resumen.html" type="text/html" hreflang="es" charset="utf-8" rel="help">Resumen HTML </a>

Efecto:
    Define un enlace para redigir a la pagina resumen.

Seccion:
    body

Etiquetas:
    <a> </a>

Atributos:
    href: indica la url, valor = http://www.e-style.com.ar/resumen.html (obligatorio)
    type: tipo de contenido del destino, valor = text/html (opcional)
    hreflang: idioma del documento destino, valor = es (opcional)
    charset: codificacion de caracteres del destino, valor = utf-8 (opcional)
    rel: relacion entre los documentos, valor = help (opcional)
